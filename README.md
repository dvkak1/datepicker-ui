Datepicker UI — roadmap.sh Challenge

A frontend UI component built as part of the roadmap.sh project-based learning path. This project focuses on creating a clean and structured datepicker interface using only HTML and CSS.

📌 Overview

The Datepicker UI is a static component that mimics the layout of a typical calendar picker used in modern web applications. The goal of this project is to strengthen understanding of layout systems, spacing, and component structuring without relying on JavaScript.

🎯 Objectives
Build a visually accurate datepicker interface
Practice CSS Grid for calendar layouts
Improve component structuring and hierarchy
Apply consistent spacing and alignment
Use pseudo-elements and positioning where appropriate
🧱 Features
Input field with calendar trigger icon
Calendar header with month and navigation buttons
Weekday labels (Sun–Sat)
Calendar grid layout (7-column structure)
Styled date cells
Visual separation of UI sections
🛠️ Built With
HTML5 (semantic structure)
CSS3 (Flexbox, Grid, positioning)
